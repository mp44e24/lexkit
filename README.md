# Contract Analysis Tool

Eine einfache Webanwendung zur automatischen Analyse von Verträgen.  
Der Text eines Vertrags wird analysiert und auf mögliche Risiken, fehlende Klauseln und wichtige Hinweise überprüft.

## Funktionen

- Analyse von Vertragstexten
- Erkennung möglicher Risiken
- Hinweise auf problematische oder fehlende Klauseln
- Anzeige der Anzahl analysierter Wörter
- Übersichtliche Ergebnisdarstellung
- Export / Kopieren der Analyseergebnisse

## Analysebereiche

Die Anwendung überprüft unter anderem:

- Haftungsklauseln
- Gerichtsstandsklauseln
- Schriftformerfordernisse
- Laufzeitregelungen
- mögliche Risiken im Vertrag

Die Analyse dient ausschließlich zur **ersten Orientierung** und ersetzt keine rechtliche Beratung.

## Beispiel Ergebnis

Die Analyse zeigt:

- Risikostufe des Vertrags
- erkannte Risiken
- Hinweise zu möglichen Problemen
- eine allgemeine Empfehlung
